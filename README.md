# 1490 Lively Ridge — Design Direction (Shareable)

Static site for sharing the California Modern Japandi design direction for the 1490 Lively Ridge renovation.

**Live site:** (deployed via Vercel)

## Contents

- `index.html` — landing page with hero grid + 9-element composite signature + nav
- `mood-board.html` — full 35-image canon, organized by room type, with MOOD-ONLY classification badges
- `images/` — 35 reference images (compressed 1800px wide JPEG q80; ~18MB total)

## What's NOT in this repo (privacy)

- Construction budget details
- Owner / family / pet specifics
- Renovation scope contracts
- Personal financial info

Those live in the private `omoji-personal/HomeAI` repo for owner reference.

## Canon designers featured

- **Cathie Hong Interiors** (Bay Area) — Campbell House is the spine project
- **OWIU Studio** (LA) — Palmero / Ryokan / Echo Park / Duane / Brentwood (NOT Biscuit Loft / Glass Ridge)
- **Sarah Sherman Samuel** — Mandy Moore's mid-century home
- **Jenni Kayne Interiors** — Lake House + Santa Ynez Ranch
- **Studio LIFE/STYLE** (Shannon Wollack + Brittany Zwickl)
- **Allprace Homes** (Shanty Wijaya)
